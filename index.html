Multi-Pixel Photon Counter (MPPC)

a solid state photodetector that uses multiple avalanche photodiode (APD) pixels operating in Geiger mode. Each pixel in the MPPC outputs a pulse at the same amplitude when it detects a photon. Pulses generated by multiple pixels are output while superimposed onto each other.
